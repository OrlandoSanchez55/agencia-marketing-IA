## Instalación en 3 pasos (5 minutos)

### Paso 1 — Instala Claude Code
Si no lo tienes, descárgalo en: https://claude.ai/download
(Necesitas una suscripción a Claude — Plan Pro $20 USD/mes)

### Paso 2 — Abre esta carpeta con Claude Code
En tu terminal, escribe:
```
claude
```
Dentro de la carpeta que descargaste.

O si usas VS Code: abre la carpeta y Claude Code se activa automáticamente.

### Paso 3 — Configura tu negocio
Escribe en el chat:
```
/configurar
```
Responde las 10 preguntas y en 5 minutos tus agentes estarán listos para tu negocio específico.

## Tus 5 Agentes

| Comando | Agente | Para qué sirve |
|---|---|---|
| `/contenido` | Agente Contenido | Carruseles, calendarios, hooks, reels |
| `/ads` | Agente ADS | Auditoría de campañas Meta, copy de anuncios |
| `/copy` | Agente Copy | Voz de marca, captions, bio, scripts |
| `/embudo` | Agente Embudo | Diagnóstico y optimización de tu funnel |
| `/filtro` | Agente Filtro | Revisa todo antes de publicar — evita bloqueos Meta |
| `/mentor` | Agente Mentor | Análisis semanal — qué hacer esta semana |

